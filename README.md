# skyblock

> 基于 LSE 的基岩版空岛插件

把"空岛玩法的几大基础设施"打包成一个开箱即用的整体 岛屿生命周期、岛屿保护、传送点、模板系统、管理员工具、第三方扩展机制。

## 特性
  - 完整空岛体系 多人同岛 / 下界岛屿 / 自定义岛屿（主城、公共区、矿区）​
  - 精细权限控制 43 个玩家行为权限 + 6 个世界事件，白名单可单独放权​
  - 木牌传送点 告示牌写 `[传送点]` 即可创建，支持公开 / 私有切换​
  - 可视化模板编辑 手持木斧选区，`/tpl` 一键导出主世界 + 下界配套 `.mcstructure`​
  - 管理员代理 (sudo) 以岛主身份临时进入他人岛屿协助维护，操作完毕一键退出​
  - 空间索引加速 O(1) 的坐标查询，1000+ 岛屿实测无卡顿​
  - 多语言支持 内置中英文，第三方扩展可热注册自己的语言包​
  - 第三方扩展友好 把 `.js` 文件丢进 `plugins/` 即自动加载​
