# 岛屿等级

::: tip 是否启用看腐竹
本页基于 `level.js` 扩展。如果你的服没有 `/is level` 命令，说明腐竹关闭了等级玩法。腐竹侧的配置见 [岛屿等级配置](/server/level)。
:::

## 等级怎么算

每种方块都有一个"价值分数"（由腐竹配置）。系统扫描你岛屿范围内的 **所有方块**，把分数累加，最后：

```
岛屿等级 = ⌊总分 / 100⌋
```

- 价值表里**没有**的方块不计分

## 计算等级

```
/is level calc
```

```
计算完成 当前岛屿等级为 47 距离下个等级还需要 28 点数
```

## 排行榜

```
/is level top
```

显示前 10 名岛屿：

```
===== 岛屿等级排行榜 =====
#1 chuxiao [chuxiao, 233] - 152
#2 b [bob] - 89
...
```

::: tip 排行榜只统计算过的岛屿
**没人执行过 `/is level calc` 的岛屿不会上榜**。你的岛屿等级也不会随建造自动更新——必须 calc 一次才记录。

如果想常驻榜首，定期 `/is level calc` 刷新即可。
:::

## 查看手持方块的价值

```
/is level check
```

任何玩家都能用。手持一个方块，查它每个值多少分：

```
minecraft:diamond_block 的价值为 100 分
```

未列入价值表的方块会提示：

```
minecraft:dirt 一文不值
```
