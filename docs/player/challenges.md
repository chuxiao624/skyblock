# 岛屿挑战

::: tip 是否启用看腐竹
本页基于 `challenges.js` 扩展。如果你的服没有 `/is challenge` 命令，说明腐竹关闭了挑战玩法。腐竹侧的配置见 [岛屿挑战配置](/server/challenges)。
:::

## 打开挑战 GUI

```
/is challenge
```

弹出 **等级列表**，每个等级用一种物品图标显示：

```
[苹果]    新手      ← 默认解锁
[木镐]    基础      ← 需要先在 "新手" 完成 5 个挑战
[铁锭]    简单      ← 需要先在 "基础" 完成 6 个挑战
[钻石]    普通      ← ...
[下界合金] 困难
```

未解锁的等级会用 **红色** 显示，点进去会提示需要的前置条件。

## 完成挑战

::: warning 必须在自己岛屿上
所有挑战的提交都需要你 **站在自己的岛屿上**。在主城或别人岛上 `/is challenge` 会提示 "你必须在自己的岛屿上"。
:::

| 类型 | 需求 |
| --- | --- |
| **物品收集** | 背包里凑齐指定物品 → 提交后**物品会被扣除** |
| **方块在身边** | 在指定半径（默认 3 格）内放置 / 留下指定方块 |
| **实体在身边** | 在指定半径（默认 10 格）内集齐指定实体 |
| **岛屿等级** | 你的岛屿等级达到指定值（需要 `/is level calc` 先算过） |
