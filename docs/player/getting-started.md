# 快速开始

## 指令

```
/is help             # 打开帮助菜单
/is create       # 创建你的岛屿
/is spawn        # 回到岛屿出生点
```

## 创建你的岛屿

输入：

```
/is create
```

## 回家

```
/is spawn
```

直接回到岛屿出生点。

想换出生点？走到岛上的某个位置（**必须在主世界 + 自己的岛上**），然后：

```
/is setspawn
```

## 查看岛屿信息

```
/is info
```

弹出一个 GUI，显示岛屿名、岛主、尺寸、成员数。

## 邀请朋友

岛主才能邀请：

```
/is invite <玩家名>
```

被邀请的玩家会收到提示，他需要在 **30 秒内** 输入：

```
/is invite accept   # 接受
/is invite refuse   # 拒绝
```

接受后他会被传送到你的岛屿出生点。

::: tip 想自己加入别人的岛
你不能"主动加入"，必须等岛主邀请你。
:::

## 离开 / 踢出

成员可以主动离开：

```
/is leave
```

岛主可以踢人：

```
/is kick <玩家名>
```

被踢的玩家会被传送回主城。
