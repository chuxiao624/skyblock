# 邀请与成员

## 邀请流程

::: tip 谁能邀请
只有 **岛主** 能邀请新成员。普通成员无法邀请。
:::

### 1. 岛主发邀请

```
/is invite <玩家名>
```

要求：

- 目标玩家**当前在线**。
- 目标玩家**没有岛屿**（自己有岛 / 在其他岛 都不行）。
- 你的岛屿成员数没达到上限。

### 2. 被邀玩家响应

目标玩家会收到一条聊天提示：

```
<你> 邀请你加入他的岛屿，输入 /is invite accept 接受，/is invite refuse 拒绝
```

他需要在 **30 秒内** 输入：

```
/is invite accept   # 接受
/is invite refuse   # 拒绝
```

30 秒未响应会自动作废。

### 3. 接受后

- 被邀玩家自动传送到岛屿出生点。
- 重生点设为岛屿出生点。
- 双方都收到加入成功提示。

## 成员上限

腐竹在 `config.json` 中配置 `member_limit`（默认 3，含岛主）。

达到上限后再邀请会被拒绝，提示 `岛屿成员已达到上限`。

## 离开

成员可以主动离开：

```
/is leave
```

离开后：

- 被传送回主城。
- 重生点重置回主城。
- 不影响你创建自己的新岛屿。

::: warning 岛主不能 leave
岛主执行 `/is leave` 会被拒绝。先 `/is transfer` 转让岛主，或者 `/is disband` 解散。
:::

## 踢出成员

岛主可以踢人：

```
/is kick <玩家名>
```

被踢的玩家：

- 立即被传送回主城。
- 收到提示。

`/is kick` **不能踢岛主自己**。
