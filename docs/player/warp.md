# 传送点 (Warp)

插件的 的传送点系统**基于木牌**——没有专门的命令去"创建"或"重命名"。

1. 创建：木牌第 1 行 `[传送点]` + 第 2 行名字 + **右键**
2. 改名：站到木牌前 **蹲下右键** 它

## 创建一个传送点

### 第一步：放一块木牌

任何木牌都行（橡木、桦木……），木牌必须 **放在你自己的岛屿上**。

### 第二步：写两行字

| 行 | 内容 |
| --- | --- |
| **第 1 行** | `[传送点]`（必须包含方括号） |
| **第 2 行** | 这个传送点的名字（≤ 16 字符，不能含 `§` 颜色码） |
| 第 3、4 行 | 随意，不影响判定 |

::: warning 标签必须一字不差
- `[传送点]` 必须是 **英文** 的方括号：`[` `]`，不是 `【` `】`。
:::

### 第三步:右键木牌

::: danger 这一步必须做！
操作：**右键** 木牌
:::

激活后：

- 木牌的 `[传送点]` 文字会变绿色（`§a[传送点]`），表示已激活。
- 你会收到提示 `传送点 <名字> 创建成功`。
- 这个传送点会出现在 `/is warp list` 中，**默认是私有的**。

## 重命名一个传送点

::: warning 第二个隐形操作
对着 **已激活的** 传送点木牌，**蹲下 + 右键** 它，会弹出一个表单让你输入新名字。
:::

操作步骤：

1. 走到木牌前。
2. **蹲下**。
3. **右键** 木牌。
4. 弹出 GUI，第一个输入框预填的就是原名字。
5. 改成新名字，点确认。
6. 木牌文字会自动更新，数据也会同步。

::: tip 区分"创建"和"改名"
- 木牌还没激活（第 1 行 `[传送点]` 还是白色）→ 蹲下右键 = **创建**
- 木牌已激活（第 1 行 `[传送点]` 是绿色）→ 蹲下右键 = **改名**

两者都是同一个手势，靠插件根据状态自动分流。
:::

## 删除一个传送点

**直接破坏那块木牌**。插件监听了木牌破坏事件，会自动清理对应的传送点数据。提示：

```
传送点删除成功
```

要破坏木牌你需要有 `destroy_block` 权限——你在自己岛上自然有。

## 公开 / 私有

新建的传送点默认 **私有**，只有岛屿成员能用。

切换公开 / 私有：

```
/is warp toggle
```

GUI 列出你岛上所有传送点，点其中一个即切换状态：

| 标签 | 含义 |
| --- | --- |
| `§a公开` | 出现在 `/is warp public` 列表里，所有玩家可用 |
| `§7私有` | 只有自己 / 岛屿成员能在 `/is warp list` 看到 |

## 使用传送点

### 自己的传送点

```
/is warp list
```

弹出 GUI，列出你岛屿的所有传送点

### 公开传送点

```
/is warp public
```
