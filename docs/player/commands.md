# 命令速查

所有玩家命令都在 `/is` 命名空间下。直接 `/is` 或 `/is help` 可以在游戏内查看帮助。

## 基础

| 命令 | 作用 |
| --- | --- |
| `/is` | 显示帮助（同 `/is help`） |
| `/is help` | 显示完整帮助文本 |
| `/is create` | 创建岛屿（选模板） |
| `/is disband` | 解散岛屿（需要二次确认） |
| `/is info` | 查看当前岛屿信息 |
| `/is spawn` | 传送到自己岛屿的出生点 |
| `/is setspawn` | 把当前位置设为出生点（仅自己岛上、仅主世界） |
| `/is transfer` | 把岛主转让给一个成员 |

| 命令 | 作用 |
| --- | --- |
| `/is invite <玩家>` | 邀请玩家加入你的岛 |
| `/is invite accept` | 接受邀请 |
| `/is invite refuse` | 拒绝邀请 |
| `/is kick <玩家>` | 踢出某个成员（仅岛主） |
| `/is leave` | 离开当前岛屿（非岛主） |

| 命令 | 作用 |
| --- | --- |
| `/is perm edit` | 编辑岛屿默认权限（GUI） |
| `/is perm allowlist` | 管理白名单玩家（GUI） |
| `/is perm allowlist <玩家>` | 编辑指定玩家的白名单（GUI） |
| `/is perm add <玩家> <节点>` | 命令式：给玩家添加单个权限节点 |
| `/is perm remove <玩家> <节点>` | 命令式：移除某个权限节点 |

详见 [权限管理](./permissions)。

| 命令 | 作用 |
| --- | --- |
| `/is warp list` | 列出自己岛屿的所有传送点 |
| `/is warp public` | 浏览所有公开传送点 |
| `/is warp toggle` | 切换某个传送点的公开 / 私有 |

设置 / 编辑传送点是 **木牌交互**，详见 [传送点](./warp)。

## 扩展玩法

| 命令 | 作用 | 依赖扩展 |
| --- | --- | --- |
| `/is challenge` | 打开挑战 GUI | `challenges.js` |
| `/is level calc` | 计算岛屿等级（5 分钟冷却） | `level.js` |
| `/is level top` | 岛屿等级排行榜 | `level.js` |
| `/is level check` | 查看手持方块的价值 | `level.js` |
| `/is level set <分数>` | 设置手持方块价值（仅 OP） | `level.js` |

## 操作

| 操作 | 触发 |
| --- | --- |
| 在自己岛上放木牌，第 1 行写 `[传送点]`，第 2 行写名字，**右键** 木牌 | 创建一个传送点 |
| 蹲下右键已激活的传送点木牌 | 编辑传送点名字 |
| 破坏传送点木牌 | 删除传送点 |
| 岛主**蹲下 + 手持指南针 + 攻击某个玩家** | 打开该玩家的权限白名单 GUI |

不要忘了这些 —— 它们没法靠 `/is help` 自己发现。
