# 岛屿管理

## 查看信息

```
/is info
```

弹出 GUI，显示：

- 岛屿名（默认是岛主的玩家名，可以通过转让岛主隐式改）
- 岛主
- 尺寸（边长 × 边长）
- 成员总数

## 出生点

每个 **成员** 都有自己独立的出生点。死亡后会回到自己的出生点（不是岛主的）。

| 命令 | 作用 |
| --- | --- |
| `/is spawn` | 传送到自己的岛屿出生点 |
| `/is setspawn` | 把当前位置设为出生点（仅在自己岛上 + 仅主世界） |

刚加入岛屿时，成员的出生点会复制岛主的当前出生点。之后可以自己 `/is setspawn` 改。

## 解散岛屿

```
/is disband
```

执行后会出现 30 秒确认窗口：

```
你确定要解散岛屿吗?请在 30 秒内再次输入 /is disband 以确认
```

在 30 秒内再次输入 `/is disband` 才会真正删除。否则确认窗口会过期，需要重新走一遍流程。

::: warning 解散次数有上限
腐竹默认配置上限 3 次。达到上限后再 `/is disband` 会被拒绝。

如果你的岛被误删 / 你被误踢，可以联系管理员请求重置解散次数。
:::

解散后：

- 你和所有成员都会被传送回主城。
- 你的解散次数 +1。

## 转让岛主

```
/is transfer
```

弹出成员列表 GUI，选择目标成员，二次确认后：

- 你从 `岛主` 降为 `普通成员`
- 目标成员升为 `岛主`
- 岛屿名会变成新岛主的玩家名

::: tip 想离开岛但是岛主
作为岛主你不能直接 `/is leave`，否则岛屿会"无主"。先 `/is transfer` 把岛主转给别人，再 `/is leave`。或者直接 `/is disband` 解散。
:::
